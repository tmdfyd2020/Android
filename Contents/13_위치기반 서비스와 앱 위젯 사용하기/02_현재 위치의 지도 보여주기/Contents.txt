- 현재 위치의 지도 보여주기 -

○ 안드로이드에서 앱 화면 안에 지도를 넣을 수 있도록 맵 프레그먼트가 제공된다.
  XML 레이아웃에 맵 프레그먼트를 추가해서 지도를 보여줄 때 필요한 과정

  1단계 : Google Play Services 라이브러리 사용 설정하기
  구글맵 서비스 v2는 Play Servcies 라이브러리를 사용한다. 따라서 구글맵을 사용하려면 이 라이브러리를 추가해야 한다.
  
  2단계 : XML 레이아웃에 맵 프레그먼트 추가하기
  앱 화면에 지도를 넣으려면 XML 레이아웃에 맵 프레그먼트를 추가한다.
  
  3단계 : 소스 코드에서 내 위치로 지도 이동시키기
  지도를 띄울 때 내 위치가 보여야 하므로 소스 코드에 지도를 내 위치에 맞추어 이동시키는 기능을 추가한다.
  
  4단계 : 매니페스트 설정 추가하기
  지도를 사용하려면 권한이 필요하다. 이 외에도 필요한 설정 정보를 매니페스트에 등록해야 한다.
  
  5단계 : 지도 API키
  구글맵 서비스를 사용하려면 구글 콘솔에서 지도 API 키를 발급 받아 앱의 매니페스트에 넣어줘야 한다.

○ 구글맵 API 키 발급받기

  https://console.developers.google.com
